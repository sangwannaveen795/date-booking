# Date Booking WhatsApp

A simple Date Booking website built with HTML, CSS, and JavaScript.

## Features

* Select Start Date
* Select End Date
* Send Booking Details to WhatsApp
* Mobile Friendly Design
* Simple and Clean UI

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

```
date-booking-whatsapp/
│
├── index.html

```

## How to Run

1. Download or Clone the Repository
2. Open `index.html` in your browser
3. Select Start Date and End Date
4. Click **Send on WhatsApp**
